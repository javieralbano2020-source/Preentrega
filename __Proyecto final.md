2. Crear un archivo “README.md” donde se indique el orden en el que se prueban las cosas y/o donde están las funcionalidades.
3. Crear un proyecto Django y al menos una aplicación Django.
4. Crear por lo menos 3 modelos.
5. Crear un CRUD para cada modelo creado.
6. Crear al menos un formulario HTML de búsqueda para consultar la base de datos y listar los elementos filtrados.
7. Usar herencia de plantillas (base.html) con `{% block %}` en templates.
8. Crear una página "/about" "acerca de mí" o nombre de URL parecido.
9. Crear una vista para Login, Logout y Register.
