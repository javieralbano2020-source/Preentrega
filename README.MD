# Sistema de Gestión de Eventos 📅

Este es un proyecto desarrollado en **Python y Django** para la gestión, registro y búsqueda de eventos, personas y ubicaciones.

## 🚀 Funcionalidades
* **Panel de Administración:** Gestión total de la base de datos desde `/admin`.
* **Registro:** Formulario para dar de alta nuevas personas.
* **Buscador:** Filtro de eventos por nombre en tiempo real.
* **Modelos:** Implementación de relaciones entre Eventos, Personas y Ubicaciones.

## 🛠️ Tecnologías utilizadas
* Python 3.14.2
* Django 6.0.4
* SQLite (Base de Datos)
